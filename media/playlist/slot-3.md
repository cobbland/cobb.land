I'm trying something different this time: The [Nuzlocke Challenge](https://m.bulbapedia.bulbagarden.net/wiki/Nuzlocke_Challenge) (with some modifications).

## Rules

Here are my slightly modified Nuzlocke Challenge rules:

- Can only (try to) catch the first pokemon encountered in each area
    - Pokemon encountered before obtaining pokeballs don't count
    - Encountered pokemon from evolutionary lines already caught don't count
- Once a pokemon faints it must be released
- Nickname all caught pokemon
- Each badge grants one "life"—a "white out" uses one life, after which all pokemon but the starter must be released
- After three failed runs, the challenge is considered failed
