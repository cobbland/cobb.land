## Colors

Press and hold one of the following button combinations while turning on the gb.

| Color Filter    | Button Press  |
| :-------------- | :------------ |
| Brown           | UP            |
| Blue            | LEFT          |
| Pastel          | DOWN          |
| Green           | RIGHT         |
| Red             | UP + A        |
| Dark Blue       | LEFT + A      |
| Orange          | DOWN + A      |
| Dark Green      | RIGHT + A     |
| Dark Brown      | UP + B        |
| Grayscale       | LEFT + B      |
| Yellow          | DOWN + B      |
| Negative        | RIGHT + B     |
| DMG 1 (Gameboy) | RIGHT + A + B |
| DMG 2 (Gameboy) | LEFT + A + B  |
