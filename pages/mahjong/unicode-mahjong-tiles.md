All of the mahjong tiles are included in unicode and can be rendered in html with the following:

| Name       | Symbol               | Unicode                                          |
| :--------- | :------------------- | :----------------------------------------------- |
| Winds      | 🀀 🀁 🀂 🀃             | `&#x1F000;`–`&#x1F003;`                          |
| Dragons    | 🀄︎ 🀅 🀆               | `&#x1F004;&#xFE0E;`<sup><a href=#1down id="1up">1</a></sup>, `&#x1F005;`–`&#x1F006;` |
| Characters | 🀇 🀈 🀉 🀊 🀋 🀌 🀍 🀎 🀏 | `&#x1F007;`–`&#x1F00F;`<sup><a href=#2down id="2up">2</a></sup>                      |
| Bamboo     | 🀐 🀑 🀒 🀓 🀔 🀕 🀖 🀗 🀘 | `&#x1F010;`–`&#x1F018;`                          |
| Dots       | 🀙 🀚 🀛 🀜 🀝 🀞 🀟 🀠 🀡 | `&#x1F019;`–`&#x1F021;`                          |
| Flowers    | 🀢 🀣 🀤 🀥             | `&#x1F022;`–`&#x1F025;`                          |
| Seasons    | 🀦 🀧 🀨 🀩             | `&#x1F026;`–`&#x1F029;`                          |
| Joker<sup><a href=#3down id="3up">3</a></sup>  | 🀪                    | `&#x1F02A;`                                      |
| Reverse    | 🀫                    | `&#x1F02B;`                                      |

<p class="note footnote" id="1down">1: The extra stuff at the end is a <a href="https://unicode.org/faq/vs">unicode variation selector</a> to force it to render as a glyph instead of an emoji.<a href="#1up">↩︎</a></p>

<p class="note footnote" id="2down">2: Note that digits go from 0 to F.<a href="#2up">↩︎</a></p>

<p class="note footnote" id="3down">3: If you use that sort of thing.<a href="#3up">↩︎</a></p>
